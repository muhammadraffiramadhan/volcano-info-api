## API Endpoints
- `/volcanoes`: Get a list of all volcanoes.
- `/volcanoes/{id}`: Get details of a specific volcano by ID.
- `/eruptions`: Get information about recent eruptions.
- `/eruptions/{id}`: Get details of a specific eruption by ID.
